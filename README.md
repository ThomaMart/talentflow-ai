# TalentFlow AI

> AI-powered recruitment platform built with WordPress, FastAPI and Docker.

TalentFlow AI is a modern recruitment platform combining the flexibility of WordPress with AI-powered services.

The goal is to demonstrate how a traditional CMS can be extended using modern software architecture, REST APIs and Large Language Models.

---

## Features

- WordPress recruitment portal
- Custom WordPress plugin
- FastAPI backend
- AI CV analysis
- Candidate scoring
- REST API
- Docker environment
- CI/CD with GitHub Actions
- Modern software architecture

---

## Architecture

```
                WordPress

           Custom Plugin
                │
                │ REST API
                ▼
           FastAPI (Python)
                │

                │
```
